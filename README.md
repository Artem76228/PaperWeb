# 🌐 PaperWeb v1.0.0

**PaperWeb** is a fast, lightweight, open-source text-based web browser designed specifically for the **M5Stack Cardputer**.

---
![](1775153371604.png)
---

## ✨ Features

- **gzip/deflate** — compressed pages work out of the box
- **UTF-8 & Cyrillic** — proper Russian/Ukrainian text rendering
- **Link navigation** — `,` and `/` to jump between links, `ENTER` to follow
- **Link highlighting** — selected link appears as cyan on black
- **Browsing history** — 10 steps back with `DEL` key
- **Zoom** — `=` and `-` (0.5x to 3.0x)
- **Smart search** — type without dots → Google search
- **WiFi Manager** — save up to 3 networks
- **Streaming parser** — displays line by line as page loads
- **Memory Guard** — never crashes on large pages

## 🎮 Controls

| Key | Action |
| :--- | :--- |
| `ENTER` | Open URL / Search |
| `TAB` | WiFi Manager |
| `,` / `/` | Previous / Next link |
| `;` / `.` | Scroll |
| `=` / `-` | Zoom |
| `` ` `` | Exit / Back |
| `DEL` | History back |

## 📦 Installation

1. Download `PaperWeb_v1.0.0.bin` from [Releases](https://github.com/Artem76228/PaperWeb/releases)
2. Flash using **M5Burner** or **ESP32 Download Tool**
